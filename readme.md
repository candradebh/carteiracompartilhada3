## Front-End

Front end com Vue

### Instalando

`npm install`
`npm run lint`
`npm run dev`


## Back-End

Back end com java usando microserviços

Executando os serviços: 

- 1 - config-server-service
- 2 - eureka-server-service
- 3 - api-gateway-service
- 4 - auth-service
- 5 - investimentos


## Adicionando submodules

git submodule add <url.git>
